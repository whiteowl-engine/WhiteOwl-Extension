What is Houdini Swap?

Houdini Swap is a non-custodial privacy transaction protocol. It uses liquidity pools and routing to break the on-chain link between sender and recipient.

How to get API keys:

  1. Go to houdiniswap.com
  2. Register as a Partner in the developer section
  3. Generate an API Key + Secret in your dashboard
  4. Paste them here

Fees: Houdini charges a small service fee (shown in the quote before sending).

Time: Private sends typically take 2–5 minutes.

Data Disclosure

Before you use WhiteOwl, please review how we handle your data:

  • ›Page content & tab URLs — read on supported crypto sites to detect tokens and provide analysis.
  • ›Axiom auth tokens — intercepted from axiom.trade requests so you can trade without leaving the panel. Stored locally only.
  • ›Tab screenshots — captured only when you press the Inspector button. Sent to your local server for AI analysis.
  • ›Chat messages — sent to your local WhiteOwl server for AI processing. Never sent to third parties.
  • ›Wallet data — private keys encrypted with your PIN and stored locally in the browser. Never transmitted.

All data stays on your machine or goes to your local server (localhost). Nothing is collected by us or shared with advertisers. Full Privacy Policy

Use of information from Google APIs adheres to the Chrome Web Store User Data Policy, including Limited Use requirements.
